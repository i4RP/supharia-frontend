# v0.1.0 — 初期リリース

**リリース日**: 2026-02-25

## 概要

React 19 + ViteからNuxt 4.2.2への完全マイグレーション。DXS Swap DEXフロントエンドの初期バージョン。

## 変更内容

### 新規
- Nuxt 4.2.2プロジェクト構造
- Tailwind CSS v4 + カスタムテーマシステム
- Pinia状態管理（AppStore, WalletStore）
- 21ページのVue SFCコンポーネント
- vue-chartjs によるチャート（Recharts代替）
- Canvas APIによるローソク足チャート
- lucide-vue-next アイコン
- モバイルナビゲーション
- ダーク/ライトモード切替
- EN/JA言語切替
- Mainnet/Testnetネットワーク切替
- ウォレット接続モーダル（Extension/WIF/Mnemonic）
- 利用規約モーダル
- トークンセレクターモーダル

### アーキテクチャ
- モジュラー・モノリス構造
- TypeScript型定義（12ファイル）
- 定数ファイル（6ファイル）によるモックデータ集約
- Composablesパターンによるロジック共有
- Nuxtファイルベースルーティング

### 技術的変更（React → Vue）
- `useState` → `ref()` / `reactive()`
- `useEffect` → `onMounted` / `watch`
- `className` → `:class`
- JSX条件分岐 → `v-if` / `v-show`
- `.map()` → `v-for`
- `onClick` → `@click`
- `onChange` + value → `v-model`
- Recharts → vue-chartjs + chart.js
- lucide-react → lucide-vue-next
- React Context → Pinia stores
- React Router → Nuxt file-based routing

## データ

すべてのデータはモック。外部API接続なし。
